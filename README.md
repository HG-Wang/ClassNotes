## 📝 使用说明

### 添加新笔记

1.  在对应分类目录下创建 `.md` 文件
    
2.  更新本 README 的笔记目录
    
3.  更新 `CHANGELOG.md` 更新日志
    

### 分类规范

```text
Notes/
├── README.md          # 本索引文件
├── CHANGELOG.md       # 更新日志
├── 数据库/            # 数据库相关笔记
├── 编程语言/          # 编程语言学习笔记
├── 系统设计/          # 系统设计笔记
└── ...
```
