**2.2　建立光控灯系统的有限状态机模型**

*光控灯是一种智能灯光控制系统，灯的开和关依据灯所在位置光照度变化而改变。当光照度低于 x 勒克斯时灯处于开状态，当光照度高于 y 勒克斯时灯处于关状态。这种光控灯系统可适用于路灯控制，也可以用于楼宇灯自动控制，以及汽车外设灯自动控制，查阅资料确定路灯控制光照度的阈值，并建立相应的自动机模型。*

答：

通过查阅资料，可以知道路灯光控系统通常在环境光照度下降到约 70 lux 时开启，并在光照度上升到 105 lux 以上关闭，以避免频繁切换。（正比例，1:1.5）
建立自动机模型：
- 系统有两个状态， 分别是开灯（On）和关灯（Off）
- 输入事件是光照强度 E , 划分成三个条件区间
	- $E_{low}$ : E<70 (光照强度低于 70 lux，即天黑)
	- $E_{mid}$ : $70 \le E \le 105$ （光照强度介于 70 lux 到 105 lux 之间，迟滞区间）
	- $E_{high}$ : E>105 (光照强度高于 105，即天亮)

状态转移规则如下表所示：

| 当前状态        | $E <70$        | $70\le E \le 105$ | $E>105$         |
| ----------- | -------------- | ----------------- | --------------- |
| $S_0$ (OFF) | 转移到 $S_1$ (ON) | 保持                | 保持              |
| $S_1$ (ON)  | 保持             | 保持                | 转移到 $S_0$ (OFF) |

**2.5 设计下面三题的 Mealy 型 FSMD 模型**

*(2)  南北东西两个方向交通路口红路灯正交控制系统：南北方向直行绿灯 40 秒，东西方向直行绿灯 30 秒，黄灯 5 秒，在直行时可以左转，右转始终是自由的。正交控制系统是指南北方向为绿灯时东西方向为红灯，南北方向为红灯时东西方向为绿灯。为了满足安全以及提高通行要求，规定绿灯到红灯的转换需经过黄灯，黄灯时间为 5 秒，而红灯直接转到绿灯。初始状态为东西和南北都是黄灯。*

由题意所示，我们需要五个状态：
- $S_0$ : 系统初始态，南北和东西都是黄灯
- $S_1$ : 南北绿灯，东西红灯
- $S_2$ : 南北黄灯，东西红灯
- $S_3$ : 南北红灯，东西绿灯
- $S_4$ : 南北红灯，东西黄灯
输出信号：
- $L_{NS}$ : 南北方向信号灯状态，取值为{G,Y,R}
- $L_{EW}$ : 东西方向信号灯状态，取值为{G,Y,R}
循环逻辑如下： $S_0 \rightarrow S_1 \rightarrow S_2 \rightarrow S_3 \rightarrow S_4 \rightarrow S_1 \rightarrow S_2 \dots$

绘制状态转移表如下表所示

| **当前状态 (Qn​)**   | **转移条件 (Guard)** | **下一状态 (Qn+1​)** | **输出动作 (LNS​,LEW​)**     | **数据路径动作 (Datapath)** |
| ---------------- | ---------------- | ---------------- | ------------------------ | --------------------- |
| **$S_0$ (初始态)**  | $t < 5$          | $S_0$            | $L_{NS} = Y, L_{EW} = Y$ | $t = t + 1$           |
|                  | $t \ge 5$        | $S_1$            | $L_{NS} = G, L_{EW} = R$ | $t = 0$               |
| **$S_1$ (南北通行)** | $t < 40$         | $S_1$            | $L_{NS} = G, L_{EW} = R$ | $t = t + 1$           |
|                  | $t \ge 40$       | $S_2$            | $L_{NS} = Y, L_{EW} = R$ | $t = 0$               |
| **$S_2$ (南北过渡)** | $t < 5$          | $S_2$            | $L_{NS} = Y, L_{EW} = R$ | $t = t + 1$           |
|                  | $t \ge 5$        | $S_3$            | $L_{NS} = R, L_{EW} = G$ | $t = 0$               |
| **$S_3$ (东西通行)** | $t < 30$         | $S_3$            | $L_{NS} = R, L_{EW} = G$ | $t = t + 1$           |
|                  | $t \ge 30$       | $S_4$            | $L_{NS} = R, L_{EW} = Y$ | $t = 0$               |
| **$S_4$ (东西过渡)** | $t < 5$          | $S_4$            | $L_{NS} = R, L_{EW} = Y$ | $t = t + 1$           |
|                  | $t \ge 5$        | $S_1$            | $L_{NS} = G, L_{EW} = R$ | $t = 0$               |
