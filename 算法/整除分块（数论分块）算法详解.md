#算法
## 一、什么是整除分块？

整除分块（也叫数论分块）是一种常用于优化求和问题的技巧，核心思想是：

> **将“相同结果的一段区间”合并处理，从而降低时间复杂度。**

典型应用场景是形如：

$$
\sum_{i=1}^{n} f\left(\left\lfloor \frac{n}{i} \right\rfloor\right)  
$$
或者：

$$
\sum_{i=1}^{n} \left\lfloor \frac{n}{i} \right\rfloor  
$$


---

## 二、关键观察（核心思想）

考虑表达式：

$$
\left\lfloor \frac{n}{i} \right\rfloor  
$$

当 (i) 增大时，这个值是**单调递减的**，并且：

👉 很多连续的 (i)，对应的值是相同的！

例如：  
当 (n = 10) 时：

| i   | n // i |
| --- | ------ |
| 1   | 10     |
| 2   | 5      |
| 3   | 3      |
| 4   | 2      |
| 5   | 2      |
| 6   | 1      |
| 7   | 1      |
| 8   | 1      |
| 9   | 1      |
| 10  | 1      |

可以看到：

- (i=4,5) 时结果都是 2
    
- ( $i=6 \sim 10$ ) 时结果都是 1
    

👉 **这就是可以“分块”的地方！**

---

## 三、如何分块？

我们希望找到一段区间 ([l, r])，使得：

$$
\left\lfloor \frac{n}{l} \right\rfloor = \left\lfloor \frac{n}{l+1} \right\rfloor = \cdots = \left\lfloor \frac{n}{r} \right\rfloor  
$$
### 关键公式：

如果当前在 (l)，设：

$$
v = \left\lfloor \frac{n}{l} \right\rfloor  
$$

那么这一段的右端点是：

$$
r = \left\lfloor \frac{n}{v} \right\rfloor  
$$

👉 这个公式非常关键，一定要记住！

---

## 四、算法流程

我们用一个循环跳跃处理每一段：

```cpp
for (int l = 1; l <= n; l = r + 1) {
    int v = n / l;
    int r = n / v;
    // 处理区间 [l, r]
}
```

---

## 五、经典例题

### 例 1：计算

$$
\sum_{i=1}^{n} \left\lfloor \frac{n}{i} \right\rfloor  
$$

### 普通做法：

```cpp
for (int i = 1; i <= n; i++)
    ans += n / i;
```

时间复杂度：**O(n)** ❌

---

### 整除分块优化：

```cpp
long long ans = 0;
for (int l = 1; l <= n; l = r + 1) {
    int v = n / l;
    int r = n / v;
    ans += (long long)(r - l + 1) * v;
}
```

时间复杂度：**O(√n)** ✅

---

## 六、复杂度分析

为什么是 O(\sqrt{n}))？

关键结论：

- 当 (i \le \sqrt{n})：值变化快，但数量少
    
- 当 (i > \sqrt{n})：值变化慢，但区间长
    

👉 不同的 (n/i) 取值最多只有约 **2√n 个**

所以总分块数是：


O(\sqrt{n})  


---

## 七、进阶应用

整除分块不仅用于简单求和，还可以扩展到：

### 1️⃣ 带权求和

 $$
\sum_{i=1}^{n} f(i) \cdot \left\lfloor \frac{n}{i} \right\rfloor  
$$



### 2️⃣ 双重求和优化

$$
\sum_{i=1}^{n} \sum_{j=1}^{n} [ij \le n]  
$$

可以转化为：

$$
\sum_{i=1}^{n} \left\lfloor \frac{n}{i} \right\rfloor  
$$

---

### 3️⃣ 配合前缀和

如果有：
$$
\sum_{i=1}^{n} f(i)  
$$

可以先预处理前缀和：

$$
S(i) = \sum_{k=1}^{i} f(k)  
$$

然后在分块时：

```cpp
ans += (S[r] - S[l-1]) * v;
```

---

## 八、常见坑点

⚠️ 注意事项：

1. **r = n / v 一定不能写错**
    
2. 循环写法必须是：
    

```cpp
l = r + 1
```

3. 注意数据范围（要用 long long）
    
4. 防止除零（一般不会发生）

```cpp
#include <iostream>
using namespace std;
int main() {
    long long n;
    cin >> n;
    long long i = 1;
    long long sum = 0;
    long long r = 0;
    while (i <= n) {
        long long v = n/i;
        r = n/v;
        sum += (r-i+1)*v;
        i = r+1;
    }
    cout << sum << endl;
}
```

## 余数求和

![](http://cdn.sealight.site/notes/202603171645844.png)

```cpp
#include <iostream>
using namespace std;

int main() {
    long long n, k;
    cin >> n >> k;
    long long i = 1, sum = 0;
    while (i <= n) {
        if (i > k) {
            sum += (n - i + 1) * k;
            break;
        }
        long long v = k / i;
        long long r = min(k / v, n);
        sum += (r - i + 1) * k - v * (i + r) * (r - i + 1) / 2;
        i = r + 1;
    }
    cout << sum << endl;
}
```
